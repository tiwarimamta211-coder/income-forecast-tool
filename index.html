📘 Istemaal Guide (Hindi)

1

Income Sources Add Karein

Apni saari income sources (salary, freelance projects, business, investments) ka naam aur monthly amount daal kar "Add Income Source" dabayein.

2

Scenario Choose Karein

Conservative (kam growth), Realistic (average growth), ya Optimistic (high growth) mein se apna scenario select karein.

3

Forecast Period Set Karein

Slider se decide karein kitne months ka forecast chahiye — 3 mahine se le kar 5 saal tak.

4

Generate Forecast Dabayein

Graph, table aur AI insights automatically generate ho jayenge based on aapke data par.

5

Save & Export

Apna forecast save kar sakte hain future reference ke liye, ya CSV mein export kar sakte hain clients ko dikhane ke liye.
